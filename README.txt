# Shmoh Almadi Catalogue Landing Page (Updated)

## What it does
- Shows the intro logo video centered on a white frame (logo fully visible).
- Automatically switches to the catalogue image.
- "View Catalogue" button opens the Google Drive PDF link.

## Files
- index.html
- style.css
- script.js
- intro2.mp4
- catalogue-page.jpg

## Edit intro duration
Open `script.js` and change `8000` (milliseconds).
- 6000 = 6 seconds
- 10000 = 10 seconds

## Edit catalogue link
Open `index.html` and change the href on the button.

## Publish
Upload ALL files to your hosting (GitHub Pages / Hostinger / Netlify) in the same folder (root).
